# E-commerce-basic-ui-
A simple and responsive e-commerce web page built using HTML and CSS only. This project displays products in a clean card layout and is ideal for beginners learning front-end web development.
         Features
Responsive product grid layout
Clean and modern UI
Product cards with image, name, price, and button
Pure HTML & CSS (no JavaScript)
Beginner-friendly structure

           Technologies Used
HTML5
CSS3
